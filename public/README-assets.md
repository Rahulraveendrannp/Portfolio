# Add these files for your portfolio

- **profile.jpg** (or .png) — Your photo; appears in the navbar before "rahul.np".  
  If missing, the letter "R" is shown instead.

- **resume.pdf** — Your resume PDF.  
  When visitors click "Resume" they get a popup; choosing "Yes, Download" downloads this file.

Place both files in this `public` folder (same folder as this file).
